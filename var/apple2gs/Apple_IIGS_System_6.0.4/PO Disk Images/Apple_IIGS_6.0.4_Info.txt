A ProDOS Order live installer image. To use, copy to CompactFlash or USB media for a CFFA3000, or open in your preferred Apple IIGS emulator.2.) ProDOS Order 800K floppy images.3.) Apple IIGS ShrinkIt (GSHK) 800K floppy archives in Binary II format. To use, download and use GSHK to decompress and create a disk.NOTE: Disks 1 through 8 or the Live Installer are required in order to install System 6.0.4 properly. Boot from Install or Live.Install to install the System Software. Attempting to upgrade a running system may result in an incomplete install.
